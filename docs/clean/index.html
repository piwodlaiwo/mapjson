1 · Paste your keys

One key per line, or paste/upload a CSV — country names, state names, postal codes, FIPS, ISO codes, misspellings welcome. Nothing is stored except anonymous resolution outcomes that improve matching.

2 · Review

Corrections you confirm here become aliases in the next index build.

3 · Use it

The cleaned CSV appends gid (mapjson's join key), the canonical name, and ISO/FIPS codes to your data. Boundary features from api.mapjson.com/v1/geo carry the same gid, so drawing a map is a dictionary lookup: feature gid → your value → a color. The build page does exactly that — drop this CSV on it and watch it become a choropleth, code included.
